== Description ==

### Extend Link

Extend Link plugin is All-In-One for WordPress post editor to manage the links, add class to link, rel nofollow, and more.

### The Features

1. Insert any attribute to the link:
    1. Rel Nofollow.
    2. Link Title.
    3. Link ID.
    4. Link Classes.
    5. Open link in a new tab.
    6. Downloadable link.
2. All-In-One.
3. Easy to use.

== Installation ==

### Installation

1. Upload 'extend-link' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Open your post editor, select any text, click on "Extend Link" button in your editor, set your link options, enjoy.
